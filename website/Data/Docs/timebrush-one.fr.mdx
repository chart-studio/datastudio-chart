### 🔥 Description 🔥

Time Brush vous permet de filtrer des données temporelles en faisant glisser ou en balayant directement une période d'intérêt.
L'affichage initial est un aperçu utile qui montre la fréquence des éléments dans chaque intervalle de temps prédéfini (par exemple, heures, jours, années) sous forme de graphique à barres verticales.
Chaque fois que vous sélectionnez une période, tous les visuels liés sont automatiquement filtrés pour afficher uniquement les enregistrements avec des attributs de temps dans la période sélectionnée.
