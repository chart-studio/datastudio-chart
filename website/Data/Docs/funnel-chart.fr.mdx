### 🔥 Description 🔥

Les graphiques en entonnoir sont un type de graphique, souvent utilisé pour représenter les étapes d'un processus de vente et montrer le montant des revenus potentiels pour chaque étape.
Ce type de graphique peut également être utile pour identifier les problèmes potentiels dans les processus de vente d'une organisation.
Un graphique en entonnoir est similaire à un graphique à barres de pourcentage empilé (source wiki).
